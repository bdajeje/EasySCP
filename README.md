# EasySCP

GUI tool for scp (Super Copy Program)

## Installation

### Librairies

Following librairies are needed:
* -lboost_filesystem 
* -lboost_system

EasySCP uses sshpass and scp behind the scene, so:
* sudo apt-get install scp sshpass

### Compilation

We use QMake.
